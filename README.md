# Amazon Clone Website

This is my first web development project.

I created an Amazon Clone homepage using **HTML and CSS**.  
This project focuses on designing the layout and user interface similar to the Amazon website.

The project includes a navigation bar, search bar, product sections, hero banner, and footer.

---

## 🚀 Features

- Amazon style navigation bar
- Search bar design
- Location and cart section
- Hero banner image
- Product category boxes
- Hover effects on products
- Footer section
- Clean layout design

---

## 🛠 Technologies Used

- HTML
- CSS
- Font Awesome Icons

## 📌 Project Description

This project is a **frontend Amazon homepage clone** built using HTML and CSS.

It is a static website and does not include JavaScript functionality such as login or add-to-cart.

This project helped me learn:

- HTML structure
- CSS styling
- Layout design
- Flexbox
- Hover effects
- Website structure
